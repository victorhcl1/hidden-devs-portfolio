# Roblox In-Game Phone System

This repository contains a complete in-game phone system made for a Roblox experience.  
It includes social feed (Chirper), live streaming, private messages, notifications and settings, all built with Luau.

## Features

- In-game phone UI with opening/closing animation
- Social feed (Chirper) with posting, likes and reposts
- Live streaming system with viewers, likes, comments and speech bubbles
- Private messaging app with chat bubbles and contact list
- Toast notifications for messages, lives and social posts
- Per-player notification settings saved on the server
